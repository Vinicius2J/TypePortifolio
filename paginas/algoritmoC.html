C,JavaScipt,PHP,Java etc... ?

C, JavaScript, PHP e Java são todas linguagens de programação populares e amplamente usadas para desenvolver aplicativos e sistemas de software.
Também não iremos muito a fundo na linguagem C, apenas para praticar a lógica de programação.

Linguagem C

C: é uma linguagem de programação de propósito geral e de baixo nível, que é usada para desenvolver sistemas operacionais, drivers de dispositivos, jogos, entre outros. C é uma linguagem de programação compilada e é conhecida por sua eficiência e desempenho.

Linguagem JavaScipt

JavaScript: é uma linguagem de programação de alto nível e interpretada, que é usada principalmente para desenvolver aplicativos da web interativos. Ele é executado no lado do cliente do navegador da web e é capaz de criar recursos avançados, como animações, efeitos visuais e interações de usuário.

Linguagem PHP

PHP: é uma linguagem de programação de script do lado do servidor, usada principalmente para criar aplicativos da web dinâmicos e interativos. O PHP é frequentemente usado em combinação com bancos de dados, como MySQL, para criar aplicativos da web robustos e escaláveis.

Linguagem Java

Java: é uma linguagem de programação de propósito geral, que é usada para desenvolver aplicativos empresariais, sistemas distribuídos, aplicativos móveis, jogos e muito mais. Java é uma linguagem orientada a objetos e é executada em uma máquina virtual Java, o que permite que o código seja executado em várias plataformas.

int hora_cinema = 20;

Explicação: define a hora de fechamento do cinema como 20:00 (8:00 pm).

int hora_atual = 20;

Explicação: define a hora atual como 20:00 (8:00 pm).

int hora_valida = hora_cinema + 30;

Explicação: define a hora limite para que o cliente possa entrar, adicionando 30 minutos ao horário de fechamento do cinema.

if-else

Explicação: a seguir verifica a hora atual em relação à hora limite:

if(hora_atual > hora_valida)

Explicação: verifica se a hora atual é posterior à hora limite. Se for, a mensagem "Passou do tempo limite! Nao pode entrar!" será exibida.

else if(hora_atual < hora_valida - 30)

Explicação: verifica se a hora atual é antes de 30 minutos antes da hora limite. Se for, a mensagem "Nem abriu a porta do cinema" será exibida.

else

Explicação: é executado se a hora atual estiver dentro do intervalo de 30 minutos antes da hora limite até a hora limite exata. Nesse caso, a mensagem "O horario esta certo! pode entrar!" será exibida.

main()

Explicação: retorna 0, indicando que o programa foi executado com sucesso.
